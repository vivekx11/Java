# Java

## Description
This repository contains Java programming examples and practice programs covering basic to advanced concepts. It includes various folders with different Java topics and implementations.

## Repository Structure
```
Java/
├── Basics/
│   ├── Variables.java         - Variables and data types
│   ├── IfElse.java            - Conditional statements
│   ├── Loops.java             - For, While, Do-While loops
│   ├── Arrays.java            - 1D and 2D arrays
│   └── String.java            - String operations and methods
├── Main.java to Main6.java    - Original practice programs
└── README.md                  - This file
```

## Contents

### Basics Folder
- **Variables.java**: Demonstrates different data types (int, double, String, boolean, char)
- **IfElse.java**: Shows conditional logic with if-else statements and grade checking
- **Loops.java**: Covers for loop, while loop, and do-while loop examples
- **Arrays.java**: Includes 1D arrays, array operations, and 2D matrix examples
- **String.java**: String methods like length(), toUpperCase(), toLowerCase(), contains(), etc.

## How to Compile and Run

1. Navigate to the desired folder or use a Java compiler
2. Compile: `javac FileName.java`
3. Run: `java FileName`

Example:
```bash
cd Basics
javac Variables.java
java Variables
```

## Requirements
- Java JDK 8 or higher
- Any text editor or IDE (VS Code, IntelliJ IDEA, Eclipse)

## Learning Path
Begin with the Basics folder to understand fundamental Java concepts before moving to advanced topics.
